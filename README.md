# Проект: [Кофеня]

Учебный проект с адаптивной вёрсткой. Код написан с фокусом на результат и понятную структуру, при этом есть зоны для улучшения — они указаны ниже.

## Реализовано

- Вёрстка по макету (мой макет из figma).
- Адаптив для мобильных и планшетов.
- Базовые интерактивные элементы (меню, кнопки, формы).

## Текущий статус и точки роста

- Адаптив работает, но некоторые мелкие отступы на узких экранах требуют доработки.
- Часть стилей можно вынести в переменные или компоненты.
- Планирую добавлять интерактив(JS)

## Технологии

- HTML5, CSS3, JS, Json, Flexbox.

## Как посмотреть

Откройте `index.html` в любом браузере.
